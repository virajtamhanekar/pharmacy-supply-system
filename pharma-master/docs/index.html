Pharmacy Management System

A web-based solution for efficient medicine inventory and sales

📌 Project Overview

This Pharmacy Management System is a PHP and MySQL-based web application that streamlines pharmacy operations. It helps manage medicines, track inventory, and handle purchases/sales with a user-friendly interface.

🚀 Features

  • Admin dashboard to manage inventory
  • Add/edit/delete medicine records
  • User registration and login
  • Search functionality for quick medicine lookup
  • Invoice generation

🛠️ Tech Stack

  • Frontend: HTML, CSS, Bootstrap
  • Backend: PHP
  • Database: MySQL
  • Server: XAMPP/Apache
