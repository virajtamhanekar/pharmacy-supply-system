Pharmacy Supply Management System

A web-based system to manage pharmaceutical supplies efficiently

💡 Project Overview

This project provides a platform to manage pharmacy inventory, supplier details, and product stock levels. It was developed as a final-year academic project.

📂 GitHub Repository

🔗 View Source Code on GitHub

🚀 How to Run

  1. Download or clone the GitHub repo.
  2. Ensure XAMPP or a similar local server is running.
  3. Place the project in the `htdocs` folder.
  4. Access via: http://localhost/YourFolderName
